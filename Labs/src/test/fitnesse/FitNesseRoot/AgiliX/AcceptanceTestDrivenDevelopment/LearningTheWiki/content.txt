!2 Exercise 1a - Basic Wiki pages
The goal of this exercise is to introduce you to the basic !-FitNesse-! Wiki actions. You will create a project Sub Wiki, learn how to 
delete, rename and define subpage links.

 1 Start Fitnesse and open the !-FitNesse-! start page.
 2 Create a start page for your Project. 
 3 Add a header and footer page displaying info regarding your project.
 4 Add the following table contents 

!3 Contents
>SprintZero
>SprintOne
>SprintTwo

The sprints should be subpages of your project page.
